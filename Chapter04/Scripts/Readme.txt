Download the files from this directory to your C:\CPBook\LabShare\Scripts folder.

Scripts are periodically updated for correction of errors and omissions as well as to keep up with differences in behavior due to version updates for Check Point, VirtualBox and Windows.

Two pairs of scripts- addcdns_step1 and addcdns_step2 are added to provide scripted execution options for "Chapter 4, Preparing Active Directory and the domain controller" section, steps 3 to 7. Step1 renaming and rebooting the VM and Step2 performing AD and DNS installation.
